# 神经网络基础：CNN / RNN（用于 RAG 知识库）

## CNN 是什么（Convolutional Neural Network）
CNN（卷积神经网络）是一类特别擅长处理“网格状数据”的神经网络，最典型的是图像（H×W×C）。
核心思想：用卷积核在局部区域共享参数提取特征，逐层从低级纹理到高级语义。

### CNN 的关键组件
1) 卷积层（Convolution）
- 使用卷积核（filter/kernel）在输入上滑动
- 输出特征图（feature map）
- 参数共享：同一个卷积核用于所有位置

2) 激活函数（ReLU 等）
- 引入非线性，使网络能拟合复杂函数

3) 池化层（Pooling，可选）
- 下采样（例如 MaxPool）
- 降低分辨率、增强平移不变性

4) 批归一化（BatchNorm，可选）
- 稳定训练、加速收敛

5) 全连接层（FC，常见在分类头）
- 将特征映射到类别 logits

### CNN 适合什么任务
- 图像分类、目标检测、语义分割
- 也可用于语音谱图、时间序列的局部模式提取

## RNN 是什么（Recurrent Neural Network）
RNN（循环神经网络）是一类用于处理“序列数据”的神经网络。
核心思想：当前时刻的输出不仅依赖当前输入，还依赖过去的隐藏状态（记忆）。

### RNN 的基本形式
- 输入：x_t
- 隐藏状态：h_t = f(W_x x_t + W_h h_{t-1})
- 输出：y_t = g(h_t)

### RNN 的问题与改进
1) 长序列训练困难：梯度消失/爆炸
2) LSTM / GRU 通过门控机制缓解长期依赖问题
- LSTM：输入门/遗忘门/输出门 + cell state
- GRU：更新门/重置门，结构更简化

### RNN 适合什么任务
- 语言建模、文本生成、机器翻译（早期方案）
- 时间序列预测、语音识别（早期方案）
（现代大模型更多用 Transformer 取代纯 RNN）

---

## 一句话对比：CNN vs RNN vs Transformer
- CNN：擅长局部模式 + 参数共享（图像/局部特征）
- RNN：擅长顺序依赖（序列），但长依赖训练困难
- Transformer：用自注意力并行建模序列依赖，训练更稳定、可扩展
